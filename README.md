This is a simple Hello World program in C++

## How to Run

### Prerequisites
- Kate text editor
- Ubuntu Linux (24.04.1)

### Steps
1. Write the hello world code in Kate text editor then save the file in the directory as Hello.cpp
2. Use the g++ (C++ compiler to run the code)
3. In Linux type:
   g++ Hello.cpp -o hello (g++ is compiler, Hello.cpp is the source file, -o hello is the output the executable file)
   ./ hello  (specify the path where the file is located)

### Output:
Hello World!!!
